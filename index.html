Pronunciation downloader

This utility software allows you to download pronunciations of different English words

Requirements

  • awk (for example, gawk)
  • curl
  • mpg123

Installation

Simply insert utility into the preferable location

Give utility rights to be performed(chmod +x words.sh in terminal)

Running

For run, just type in terminal in folder where program is located:./words.sh

List of words can be provided by creating file list.txt in directory where downloader located.

Each word will be looked for in Cambridge Dictionaries

The result of searching will be printed in result.txt

Possible situations:

  • English and American pronunciation were found: "word OK" will be printed in result.txt
  • Only English pronunciation was found: "word UK" will be printed in result.txt
  • Only American pronunciation was found: "word US" will be printed in result.txt
  • Neither English nor American pronunciation were found: "word EMPTY" will be printed in result.txt
